# Budget & Expense Tracking Application - Resume Bullet Points (ATS-Optimized)

## Four Key Points for Resume:

1. Developed a full-stack web application using React, JavaScript, Node.js, Express.js, and MySQL database implementing user authentication, transaction management, and real-time financial analytics dashboard with interactive data visualizations using Recharts library including line charts, pie charts, and bar charts

2. Built a RESTful API using Express.js and Node.js with over 10 endpoints for CRUD operations managing transactions, budgets, financial goals, accounts, categories, tags, and savings accounts, implementing middleware-based authentication and MySQL database integration with SQL queries for data persistence and retrieval

3. Created a React frontend application using React Router for client-side routing, featuring a responsive dashboard with data visualization that displays income, expenses, savings, and budget allocations across multiple accounts using Axios for HTTP requests and API integration, enabling users to track financial goals and monitor spending patterns

4. Designed and implemented a scalable three-tier architecture separating frontend React application using Vite build tool and backend Node.js Express server, utilizing REST API principles, async await patterns, and CRUD operations to manage complex financial data relationships with proper error handling, data validation, and MySQL database schema design
